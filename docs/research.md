# 竞品研究总结

## GitHub 项目参考

1. **Pomofocus** - 极简单列布局，大号居中倒计时，任务与计时器强关联
2. **Tomodoro** - 原生 JS PWA，Picture-in-Picture 计时器，四主题+自定义强调色
3. **Pomotix** - SVG 圆形进度环，礼花动画奖励，内置环境音
4. **vue-pomo** - Vue 2 + Vuex，GitHub 风格热力图统计
5. **TwentyFive** - 极简零配置，即时可用

## 通用 UI 模式
- 居中大号倒计时（核心视觉元素）
- 模式标签（专注/短休/长休）
- SVG 圆形进度环（比条形更易读）
- 暗色/亮色主题 + 持久化
- 任务-计时器集成（先选任务再开始）
- 自动开始过渡（保持心流）
- PWA 离线优先
- 完成奖励动画（五彩纸屑）
- GitHub 风格热力图

## Bilibili 中文市场趋势
- 极简+高颜值（首要评判标准）
- 治愈系美学：柔和色彩、舒缓动画
- 免登录、浏览器即用
- 高度可定制（字体、颜色、主题）
- 桌面小组件趋势

## 推荐技术栈
- 原生 JS + CSS + LocalStorage（零依赖，极致轻量）
- Web Workers 处理计时器精度
- Service Worker + Web Manifest 实现 PWA
- 红色/番茄色作为强调色
- 暗色背景为主，暖色点缀
